<static> khtml.maplib.geometry.bounds.createFromMultiplePoints(pointArray)
(Static helper) create a {khtml.maplib.geometry.Bounds} from a point array. Problem: You cannot initialize an empty bounds and then add all points, because then it contains the coordinates 0,0. Solution: This function creates a bounds at the coordinates of the first two points and enlarges it afterwards with the remaining points.
Parameters:
pointArray
Returns:
null if arguments doesn't less than 2 points. Otherwise {khtml.maplib.geometry.Bounds}
